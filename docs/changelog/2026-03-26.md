## 詳細変更内容

### 1. 自動投稿パイプラインの土台作成
- `scripts/auto-post-blog.ts` を新規作成。
- Gemini 2.5 Pro（Google検索のグラウンディング有効化）を用いて、テーマごとに最新のWeb検索に基づいた「リサーチ」を行わせる仕組みを追加。
- リサーチ結果を「洗練された手紙風（Elegant Letter Style）」でMarkdown執筆させる処理を実装。

### 2. 「医療広告ガイドライン」自動検閲の導入
- 自動投稿スクリプト内に、執筆した記事を再度AIで読み込む「レビュー関数」を実装。
- **検閲基準**: 「最高・絶対」といった誇大表現の抑制、VOGUE風の洗練されたトーンの維持、指定フォーマット（手書きサインの付与など）が守られているかを厳格にチェックし、問題があればAIが自己修正。

### 3. 本番公開前の「承認フロー（Slack + GitHub PR）」の立案
- 自動生成された記事が直接本番に公開されるリスクを防ぐため、将来的なフローを設計。
- AIスクリプトが「下書き（Pull Request）」を作成し、Slackに通知。人間の最終確認（Approveボタン）をもって初めてVercel本番公開される安全な仕組みの枠組みを決定。

### 4. マルチプラットフォーム コンテンツ生成プロンプトの作成
- `prompts/multi-platform-content-prompts.md` を新規作成。
- Deep Researchで得た1つの深い知識を、各SNSの目的に沿って最適化するためのAIプロンプト。
  - **X (Twitter)**: 短く・フックが強い（拡散重視）
  - **Instagram カルーセル**: ページで区切り、箇条書きで分かりやすいマニュアル形式（保存重視）
  - **Instagram / YouTube リール**: 最初の3秒の視覚フックにこだわった、15〜30秒の短尺動画用台本（リーチ重視）
