# 公式LINE 初期設定テキスト集

このドキュメントは、LINE Official Account Manager（PC版またはアプリ）にコピペしてそのまま使えるテキスト素材です。Lステップ等の有料ツールを使わず、無料の標準機能で「スモールスタート」するための設定です。

## 1. アカウント設定

- **アカウント名**
  `Dr. みやか | The Skin Atelier`
- **ステータスメッセージ（ひとこと）**
  `美容皮膚科医。あなた本来の透明感を引き出す、パーソナルな美肌治療とインナーケア。`

## 2. 挨拶メッセージ（友だち追加時）

*ユーザーがLINEを通行した瞬間に自動で送られるメッセージです。「パーソナル美肌診断」へ誘導することが目的です。*

```text
はじめまして。
美容皮膚科医の みやか です🕊️
友だち追加ありがとうございます！

こちらの公式LINEでは、
Webには書ききれないスキンケアの裏話や
先行予約のシークレット案内をお届けします。

━━━━━━━━━━━━━━
🌿 まずはあなたの肌の「現在地」を
━━━━━━━━━━━━━━
お一人おひとりのお顔立ちや肌質に合った
引き算の美容医療をご提案するために、
まずは【無料パーソナル美肌診断】をお試しください。

👇以下のメニューの「診断」をタップして
いくつかご質問にお答えいただくと、
いま本当に必要なケアの方向性をお伝えします。

ご相談やご予約も、こちらのチャットから
そのまま話しかけてくださいね☺️
```

## 3. 自動応答メッセージ（パーソナル美肌診断のトリガー）

*リッチメニューの「無料パーソナル診断」をタップしたときに、自動で返信するメッセージの設定です。*

**設定方法:**
1. 応答メッセージの設定で、キーワードを「美肌診断」にする。
2. 以下のテキストを自動返信に設定する。

```text
パーソナル美肌診断へようこそ✨
最適なケアをご提案するため、以下の3つの質問に【番号】でご返信ください。

① 一番気になっているお悩みは？
1. シミ・くすみ・赤み
2. 乾燥・小ジワ・ハリ不足
3. たるみ・輪郭のもたつき
4. ニキビ・肌荒れ

② ダウンタイム（顔の赤みや腫れ）の許容度は？
A. 全く取れない（明日からメイクしたい）
B. 数日ならOK（週末だけなら）
C. 1週間程度ならOK（しっかり治したい）

③ ご年齢層を教えてください。
（例：30代前半、40代など）

【回答例】
① 1
② A
③ 30代後半

ご返信いただきましたら、みやか本人が確認し、あなたに合った美容医療・インナーケアの方向性を直接アドバイスいたします🕊️（※手動で返信するため、お時間をいただく場合がございます）
```

*(※ユーザーから回答が送られてきたら、スタンプや手動メッセージで「〇〇さま、ありがとうございます。〇〇のお悩みですね。それであれば当院のIPL治療が…」と個別にチャットを返します。これが最強の顧客リスト（カルテ）になります。)*

## 4. 予約の対応（手動チャット予約）

ユーザーがリッチメニューの「予約・相談」をタップした際の自動応答メッセージです。
キーワード: 「予約希望」などで設定します。

```text
ご予約・ご相談へ進んでいただきありがとうございます✨
以下のフォーマットをコピーして、ご希望をお知らせください。

----------
【お名前（フルネーム）】
【ご希望のメニュー（決まっていれば）】
【ご希望の日時（第3希望まで）】
第1希望：〇月〇日 〇時〜
第2希望：〇月〇日 〇時〜
第3希望：〇月〇日 〇時〜
【お悩みやご相談事項】
----------

確認次第、折り返しご連絡いたします🕊️
```
