# 洗練された美容カルーセル投稿 設計・実装指示書 (Remotion Still用)

## 1. 全体デザインシステム＆基本ルール

**コンポジションサイズ**: 1080px × 1350px (アスペクト比 4:5 / Instagram最適縦長サイズ)

### カラースキーム（提供HTMLに準拠）
- **背景（Surface）**: `#fcf9f8` (温かみのあるオフホワイト)
- **見出し・メイン（Primary）**: `#5f5e5e` (抜け感のあるダークグレー。真っ黒は絶対に使わない)
- **ラベル・アクセント（Secondary）**: `#6e5b4d` (上品なモカブラウン)

### 1. タイポグラフィと「役割分担」の徹底（和欧・明朝ゴシック混植）
すべてをおしゃれな細い明朝体にすると読解ストレスになります。フォントの役割を明確に分けます。

- **飾る英語（テクスチャ）**: `Manrope` (Light)。極小サイズ（10px等）で字送りを極端に広く（`tracking-widest`）し、画面の四隅などに配置して「雑誌のフレーム」を作る。
- **惹きつける見出し（情緒）**: `Shippori Mincho` または `Noto Serif JP` (細字〜標準)。少し大きめに配置し、美しさと権威性を担保。
- **読ませる本文（機能）**: 成分や手順などの具体的ノウハウは、視認性の高いモダンゴシック体（`Zen Kaku Gothic New` または `Noto Sans JP` の細字〜標準）を使用し、明朝体の世界観を邪魔しないスッキリとしたゴシックを選ぶ。

### 2. 余白と黄金比・日本語の「抜け感（行間）」
- 画面の周囲（上下左右）には常に最低 **80px〜100px** のマージン（セーフエリア）を確保。
- **【超重要】** 日本語本文の文字間は「標準（`tracking-normal`）」のままとし、代わりに**行間（line-height）を通常よりかなり広く（1.7〜1.8倍程度 / `leading-relaxed` や `leading-loose`）**設定し、読みやすさと大人の余裕を演出する。

## 2. 【超重要】チャンク（文節）改行の実装仕様

洗練された日本語タイポグラフィを保つため、デバイスやCSSに依存する「自動改行（単語の途中での泣き別れ）」は**厳禁**です。RemotionのPropsには、テキストを**「チャンク（意味のまとまり）の配列」**として渡し、React側で確実に改行処理を行ってください。

```javascript
// ❌ 悪い例（どこで改行されるか予測不能）
const title = "細胞レベルで変わる大人の本質スキンケア";

// ⭕️ 良い例（チャンク配列として渡し、mapで <span> 等で囲んで block 表示する）
const titleChunks = ["細胞レベルで変わる、", "大人のための", "本質スキンケア"];
```

## 3. スライド構成と文字数・レイアウト指示（全10枚）

### 🖼️ Slide 01: サムネイル（表紙 / Hook）
- **役割**: タイムラインでスクロールを止めさせる圧倒的な美しさと権威性。
- **レイアウト**: 黄金比グリッド。画面の大部分に画像を配置し、タイトルは余白に美しく配置。
- **文字数制限**:
  - タイトル：1行最大10〜12文字 × 最大3行（2〜3チャンク）。
  - 英字ラベル：極小サイズで3〜5単語（例：`Edited by Dr. Miyaka — Vol. 01`）。
- **画像指示**: カラー写真をそのまま使わず、grayscale opacity-90 などのフィルターをRemotion側で適用し、くすんだトーンに統一する。

### 📖 Slide 02: 導入（共感 / Empathy）
- **役割**: 読者の悩みを代弁し、「私のことだ」とスワイプする理由を与える。
- **レイアウト**: 背景色を少し落とす（`#f6f3f2` 等）。左寄せでテキストを配置し、右側の余白をたっぷりとる。画像はなし。
- **文字数制限**:
  - 見出し：最大15文字（例：「高い美容液でも、乾燥する。」）
  - 本文：1行15〜20文字 × 最大3行（チャンク改行必須）。

### 💡 Slide 03: コンセプト提示（Overview）
- **役割**: 紹介するノウハウの根底にある「哲学（Why）」を語る。
- **レイアウト**: 小さめの画像とテキストの組み合わせ。1pxの極細ライン（bg-outline-variant）で知的さを演出。
- **文字数制限**: 見出し15文字以内、本文は最大60文字以内。

### 🔬 Slide 04〜07: 本編（メソッド・ノウハウ） / 【雑誌度 30% : 情報 70%】
- **役割**: 「読者のメリット（情報）」を優先し、「保存」される理由となる有益な情報（How-to）を提供する。
- **レイアウト**: 
  - 背景の色味や四隅の英字ラベルで「雑誌の枠組み」は残しつつ、メイン領域は**「高級デパコスの見やすい取扱説明書」**のように整然と読みやすくレイアウトする。
  - 長文を禁止し**「小見出し」＋「1pxの極細の区切り線」＋「3行以内の短いテキスト（または箇条書き）」**のセットでモジュール化して構成する。
- **装飾ルール**:
  - 野暮ったい黒丸（・）や絵文字（✅）の代わりに、美しいセリフ体の数字「01, 02」や、極小のひし形（✦）などを使用する。
  - **「静かな強調（ハイライト）」**: 赤字や黄色のマーカーは禁止。強調はブランドカラー（`#6e5b4d`など）の文字色変更か、文字下の**極細（1px）アンダーライン**のみとする。
- **文字数制限（1枚あたり）**:
  - 小見出し：15文字以内
  - 本文：最大70文字（1行20文字 × 3〜4行以内）。これ以上は「読むのが面倒」と思われ離脱されるため削ぎ落としが命。

### 📝 Slide 08: まとめ（Takeaway / To-Do）
- **役割**: 情報の復習。ここが最も「保存（スクショ）」されるページになります。
- **レイアウト**: 画像はなし、または透明度5%程度のテクスチャのみ。ノートのように美しい箇条書き（チェックリスト）デザイン。
- **文字数制限**: 3〜4項目。各項目15〜20文字以内。

### 🕊️ Slide 09: マインドセット・エピローグ
- **役割**: 情報から「感情」へシフト。読後感を高め、ブランドの世界観への共感を促す。
- **レイアウト**: ここだけ背景色を微かに変える（例：`#f0ece8` の温かみのあるベージュ）。余白を最大（80%以上）に取り、中央にポツンと美しい言葉を配置。
- **文字数制限**:
  - メインメッセージ：15文字 × 2行以内（例：「10年後の自分のために、/ 今日、少しだけ肌を慈しむ。」）

### 📌 Slide 10: CTA（行動喚起）
- **役割**: 「保存」と「フォロー」を促す。
- **レイアウト**: 提供HTMLのSlide 10のUIを完全踏襲。線画の洗練されたアイコン（bookmark, add_circle）を使用。
- **文字数制限**:
  - メインコピー：最大15文字（例：「見返せるように保存してください」）
  - アイコン下の文字：`SAVE`, `FOLLOW` の極小英字のみ。

## 4. プロの視点からのアドバイス（スライドごとの「雑誌度」の緩急）

全10枚を同じテンションで作る必要はありません。読者の心理に合わせてレイアウトの目的を変えます。

- **【雑誌度 100% : 情報 0%】（表紙・余韻・CTA）**
  - **Slide 01, 09, 10**: VOGUEに振り切る。文字を極限まで減らし、余白と写真の力だけで「このアカウントはセンスがいい」と直感させる。
- **【雑誌度 30% : 情報 70%】（本編ノウハウ）**
  - **Slide 04〜08**: 中身に入ったら読者のメリット（情報）を優先。高級コスメの説明書のように可読性を極限まで高める。（※詳細はSlide 04〜07の項目参照）

## 5. 💡 Remotion / デザイナーへの追加指示文（コピペ用プロンプト）

Remotionのコンポーネント実装の際は、以下の要件をAIやエンジニアにそのまま渡して構築させてください。

> **【UIと可読性の両立に関する追加要件（エディトリアルUIの徹底）】**
> 全体のトーン＆マナーは「VOGUEのような海外誌」をベースにしつつも、SNSでの「情報伝達力・スマホでの読みやすさ」を損なわないよう、以下のハイブリッド設計を厳守してください。
> 
> 1. **タイポグラフィの使い分け**
>    - タイトルや英字装飾は「明朝体・セリフ体」を使用し、情緒的価値を担保すること。
>    - ハウツーや説明文などの情報ブロックは、可読性を最優先し「モダンゴシック体（細字〜標準）」を必ず使用すること。
> 
> 2. **日本語のレイアウト調整**
>    - 日本語本文の \`letter-spacing\` は広げず標準とし、代わりに \`line-height\` を \`1.7\` 以上（\`leading-relaxed\`等）に設定し、縦の余白で洗練さを出すこと。
>    - 長文のベタ打ちは避け、情報は「見出し」＋「1pxの区切り線」＋「短いテキストや箇条書き」のUIコンポーネント単位で構成すること。
> 
> 3. **スライド間の緩急（情報のメリハリ）**
>    - Slide 01（表紙）とSlide 09,10（CTA）は「ビジュアル（エモ）重視」、Slide 04〜08のノウハウ部分は「情報伝達重視」とし、高級コスメのパンフレットのように整理された分かりやすいレイアウトにすること。
> 
> 4. **ノイズの排除**
>    - 原色のハイライトや過度なドロップシャドウは禁止。強調は「ブランドカラー（Secondary等）への文字色変更」または「1pxのアンダーライン」で行うこと。
