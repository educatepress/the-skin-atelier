# Google Stitch — The Skin Atelier デザインレビュー・プロンプトセット

## 使い方

1. [stitch.withgoogle.com](https://stitch.withgoogle.com) を開く
2. 新しいキャンバスを作成
3. 以下のプロンプトを入力 or 音声(Voice Canvas)で指示

以下のブランドガイドラインに基づいて、美容皮膚科クリニックのLPをデザインしてください。

- 黄金比のような自然な美、真実の美
- 「やりすぎない洗練」——引き算の美学

【カラーパレット】
- 背景: #FDFCFA（白大理石）
- セカンダリ背景: #FAF7F2（ウォームマーブル）
- アクセント: #E8D3C9（ピンクゴールド）
- ホバー: #D4B8A9（ディープピンクゴールド）
- CTA背景: #F5E6D8（シャンパン）
- 見出し: #4A4238（ディープモカ）
- 本文: #7A7067（ソフトグレー）
- 補助: #A39A90（ミュートグレー）

【タイポグラフィ】

- 日本語本文: Noto Sans JP Weight 400

## 📐 セクション別デザインプロンプト

### 1. Hero（ファーストビュー）

```
高級美容皮膚科のHeroセクションをデザインしてください。

【コピー】
- キャッチ: 「健やかさを、肌の輝きへ。」
- サブ: "THE SKIN ATELIER BY DR. MIYAKA | HIROO"
- ボディ: 「かつて立ち止まった私だから、できること / 鏡の前でもどかしさを感じていた日々。…」
- CTA: 「扉を開ける →」
- タグ: 「— 広尾 完全招待制プライベート・アトリエ —」

【デザイン要件】
- フルスクリーン (100vh)、中央揃え
- 白大理石の背景に淡いラジアルグラデーション
- キャッチコピーは大きく（clamp 1.6rem〜3rem）
- 「輝き」の2文字だけピンクゴールドカラー
- 上部に薄いピンクゴールドの水平グラデーション線
- ボタンは白背景+ピンクゴールドボーダー、ホバーでシャンパン充填
- 余白はたっぷり。VOGUEの表紙のような、呼吸のあるレイアウト
- パララックス効果あり（スクロールで文字がゆっくり上に移動しフェードアウト）
```

### 2. Philosophy（先生のストーリー）

```
美容皮膚科医のフィロソフィーセクションをデザインしてください。

【コピー】
- タイトル: 「まっさらな私に、還るとき。」
- サブタイトル: "A QUIET RETURN TO YOUR TRUE SELF"
- 本文: 都会の喧騒を離れた広尾のアトリエ、先進の技術と調和、先生のストーリー
- 引用: 「まっさらな自分に還れる場所を、つくりたい」
- 署名: 手書きカリグラフィ画像 "Miyaka Sato, M.D." + 「美容皮膚科学会所属」

【デザイン要件】
- 最大幅768px、センター配置
- 左ボーダーのブロック引用（ピンクゴールド + シャンパングラデーション背景）
- 手書きサイン画像を右寄せに配置
- 段落間の余白を十分に取り、「手紙を読む」ような体験
- テキストは0.9rem、行間 2.3
```

### 3. Treatments（施術メニュー）

```
美容皮膚科の施術メニューセクションをデザインしてください。4つの施術があります。

【メニュー】
1. フォトフェイシャル（IPL） — Light Therapy
2. エレクトロポレーション — Infusion
3. ヒアルロン酸・ボトックス注入 — Injectable
4. 分子栄養学（インナーケア） — Inner Care

【デザイン要件】
- 2×2グリッドレイアウト（デスクトップ）、1列（モバイル）
- 各カードはシャープエッジ、薄い1pxボーダー（マーブルベイン色）
- ホバーでボーダーがピンクゴールドに変化
- カードの左上にカテゴリ名（英語、小さいCormorant Garamond）
- 各カードに左ボーダー付きの「先生のひとこと」引用
- VOGUEの見開き特集のようなエディトリアルレイアウト
- ホバーでシャンパングラデーションがカード全体にふんわり出現
```

## 📤 Stitchの出力をAGに渡す方法

1. Stitch上で気に入ったUIが出たら **HTMLエクスポート**
2. そのHTML/CSSのレイアウト構造やカラー指定を **このチャットに貼り付け**
3. AGがNext.js + framer-motion のコンポーネントに変換して反映

もしくはStitchの画面を **スクリーンショット** で共有いただければ、
そのデザインを再現するコードを書きます。
