# X (Twitter) バイラルパターン集 — Dr. Miyaka ブランド準拠

Skin Atelier のXフォロワーを伸ばすための**5種類の投稿パターン**。  
すべて「シルクトーン（上品・医学権威・引き算）」を維持しつつ、Xアルゴリズムで伸びる設計。

---

## 共通ルール（全パターン必須）

### トーン
- 一人称は「私」または「わたし」（「僕」「俺」禁止）
- 絵文字は最小限（😭😌✨💗🚩）多くても1ツイートに2個まで
- 「AI定型文」禁止: 「必見」「驚きの事実」「〇〇すると、なんと！」は絶対NG

### 医療広告法準拠
- 「絶対」「最高」「No.1」「確実に治ります」禁止
- 効果には個人差がある旨を示唆する余白を残す
- 他クリニック・他医師を名指しで貶めない

### ブランド不変則
- 「白金高輪」「広尾」等の地名禁止
- 「開業」「オープン」「当院」等のクリニック示唆禁止
- 「分子栄養学」「オーソモレキュラー」禁止（→ インナーケア・インナーウェルネス）
- 具体的な価格・料金・キャンペーン禁止
- 架空の症例・患者エピソード禁止

### X固有ルール
- 1ツイート最大 140文字（全角換算、日本語）
- Threadの場合は最大 5ポスト
- 末尾に `---` 等の区切り不要（生テキスト）
- ハッシュタグは 2-4個まで（hashtag-strategy.md 参照）

## パターン A: 神話崩しツイート 🔥
**用途**: 拡散力最強。権威性 × 逆説で「え、そうなの？」を引き出す。

### 構造
```
[権威宣言] 美容皮膚科医として、正直に言わせてください。
[神話否定] "[よくある誤解]" は、実は[意外な真実]です。
[エビデンス/理由] (1-2行で論文名や機序を軽く)
[引き算メッセージ] 何を足すかより、[ブランドらしい視点]を。
```

### 具体例
```
美容皮膚科医として、言わせてください。

「10万円の美容液」より「2000円の日焼け止めを毎朝塗ること」のほうが、
10年後の肌を決めます。

RCT (Hughes 2013) で、日焼け止めだけで光老化が24%抑制されたデータも。

何を足すかより、何を守るかを。

#美容皮膚科 #スキンケア #引き算の美学
```

### このパターンで使えるトピック
- 高価な化粧品 vs 安価な日焼け止め
- 10ステップ vs 3ステップ
- シートマスク毎日の落とし穴
- コラーゲンサプリの効果の限界
- 「効く成分」と「効かない成分」の真実

## パターン B: 論文引用ツイート 📚
**用途**: 権威性を蓄積。「信頼できる情報源」と記憶される。

### 構造
```
[主張] [肌/美容に関する一言]
[根拠] ~年に[雑誌名]に掲載されたRCT（またはメタアナリシス）で、
       [参加者数/期間]を追跡した結果です。
[PMID表記] PMID: xxxxxxxx
[学びのラップ] だからこそ、[行動変容を促すひとこと]。
```

### 具体例
```
「毎日の日焼け止めで光老化が抑制される」

これは感覚論ではなく、2013年に Annals of Internal Medicine に
掲載されたRCTで、903人を4.5年追跡した結果です。

PMID: 23732711

「効果あるの？」と疑われがちな日焼け止めですが、
エビデンスは強固です。今日から、絶やさずに。

#美容皮膚科 #エビデンスベースビューティー
```

### 使える論文バンク（この記事の参考文献から随時引用）
- Hughes 2013 (PMID: 23732711) - Sunscreen photoaging RCT
- Camargo 2021 (PMID: 34224576) - Botox Cochrane
- DiBernardo 2016 (PMID: 27363767) - IPL Review
- Kyriazidis 2023 (PMID: 37563436) - HA filler safety SR
- Ayatollahi 2020 (PMID: 32026164) - HIFU meta-analysis
- Liu 2023 (PMID: 36838636) - Ergothioneine
- Hussain 2025 (PMID: 40465501) - Cleanser & TEWL
- Celebi 2022 (PMID: 35108405) - Epithelial barrier/pollen

**【最重要】PMID は上記バンクから、または確実に存在確認できたもののみ引用。捏造絶対禁止。**

## パターン C: 対比ツイート ⚖️
**用途**: 保存されやすい。視覚的にわかりやすい「やってた / やめた」対比。

### 構造
```
❌ [やりすぎ・やってしまいがちな行動]
✅ [正しい・引き算の行動]

[1-2行の解説]

#ハッシュタグ
```

### 具体例
```
❌ 化粧水→美容液→乳液→クリーム→アイクリーム→パック
✅ 日焼け止め(朝) + 保湿(夜) + レチノール(週2〜3)

10ステップの韓国式ケアに疲れたら、
「引き算」を試してみてください。
肌は、"足す"よりも"整える"で応えます。

#美容皮膚科 #引き算スキンケア
```

### 使える対比トピック
- スキンケアのステップ数
- シミ対策の方法（レーザー vs IPL）
- ボトックスの量（多量 vs 引き算）
- 洗顔のゴシゴシ vs 泡で優しく
- 高級コスメ vs 成分シンプル

## パターン D: 告白ツイート 💌
**用途**: 共感 → フォロー転換率最高。個人の物語で記憶される。

### 構造
```
[時代感のある個人体験] ~代の頃、私は[悩み]でした。
[試行錯誤] [何をしたか、どう失敗したか]
[転換点] でも、[気づき/変化]
[今のメッセージ] [読者に贈る短い言葉]
```

### 具体例
```
20代の頃、私はひどいニキビだらけでした。

高い化粧品、流行の成分、全部試した。
でも、治らなかったんです。

変えたのは、意外とシンプルな3つだけ。
日焼け止め、保湿、睡眠。

40代の今、人生で一番肌が好きです。
遠回りしてほしくないから、書き続けます。

#美容皮膚科医の本音 #肌育
```

### 告白トピック例（すべて自分の経験ベース、架空禁止）
- 20代のニキビ期
- 高級化粧品への投資で得たこと、失ったこと
- 医師になってから気づいた美容情報のノイズ
- レチノールの使い方の間違いと正解

## パターン E: Q&A即答ツイート 💭
**用途**: 保存 + リプライ誘発。「知りたい答え」を先に出すので完読率高い。

### 構造
```
Q. [読者が検索しそうな質問]

A. [結論を最初に1行]

[理由・背景 2-3行]
[行動アドバイス 1行]
```

### 具体例
```
Q. レチノール、何歳から始めていい？

A. 20代後半からで十分間に合います。

早く始めるより「続けられる」が大切。
週1回・低濃度(0.1%前後)からスタート。
肌が慣れたら徐々に頻度を上げる「レチノール育て」で。

秋冬の乾燥期スタートは避けましょう。

#レチノール #美容皮膚科医のつぶやき
```

## エンゲージメント誘発パターン（全タイプで末尾にオプション追加可能）

### 軽いリプライ誘発（Comment Bait）
- 「あなたはA派？B派？コメントで教えてください」
- 「気になる方は、♥ つけてね。もう少し深掘りします」
- 「みんなは何使ってる？教えて〜」

### 保存誘発（Save Bait）
- 「保存しておくと、迷わなくなります」
- 「ブックマーク推奨🚩」
- 「後で見返す用に📌」

### シェア誘発（Share Bait）
- 「同じ悩みの友達に、シェアしてあげてください」
- 「気になる人いる方は、教えてあげて🌸」

**使いすぎ注意**: 1ツイートに1種類だけ、3ツイートに1回程度の頻度。毎回だとスパム感。

---

## 禁止集（違反で即削除）

- 「絶対治ります」「これさえあれば」「No.1」（医療広告法違反）
- 「クリニックに来てください」「ぜひご相談を」（集患示唆）
- 「今なら○○円」「キャンペーン中」（価格示唆）
- 架空の患者/症例/シチュエーション
- 他の医師・クリニック名指しでの批判
- PMID の数字を確証なく記載（捏造防止）
