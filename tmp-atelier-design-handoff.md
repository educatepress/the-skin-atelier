## 3. 🎨 Skin Atelier が求めるデザイン要件（Design Guidelines）
美容皮膚科・ハイエンドクリニックにふさわしい、以下のデザイン方針（Aesthetics）を厳格に守ってスタイリングしてください。

1. **洗練されたカラーパレット**
   - Book版の「ピンク/オレンジ」は禁止。
   - `hiroo-open` のWebサイトで使用されているような、**白、淡いグレー、チャコールグレー、ディープブルー、ゴールド** などの洗練されたモダンな配色。
2. **Glassmorphism (グラスモーフィズム)**
   - 背景のPexels動画を活かすため、すりガラスのような半透明の背景ぼかし（`backdrop-filter: blur(10px)` や白・黒の半透明レイヤー）を用いたテキストボックスを多用すること。
3. **優雅なタイポグラフィとアニメーション**
   - フォントは品格のあるもの（例: 明朝体や、細身でモダンなゴシック、Google Fontsの 'Inter', 'Playfair Display', 'Cormorant' など）。
   - ポップで激しいアニメーション（バウンス等）は避け、Remotionの `spring` や `interpolate` を用いた「ゆっくりと優雅なフェードイン、シームレスなスライド」を実現すること。
4. **Infographicシーンのモダン化**
   - `InfographicScene.tsx` では、Recharts等のグラフが描画されます。このグラフの色味もシックで洗練されたものにオーバーライドすること。

## 4. ⚠️ 次期AIアシスタントへの絶対厳守ルール（Critical Instructions）
- **データの「タイミング同期ロジック」は絶対に破壊しないこと。**
  動画は ElevenLabsの音声JSON（`synced-data.json`）の各単語の `start` / `end` ミリ秒と、Remotionの `useCurrentFrame()` や `useVideoConfig()` を照らし合わせてミリ秒単位で同期しています。見た目（CSS）やアニメーションのみを改修し、コアロジックを削除しないでください。
- **コンポーネント化の徹底**
  コードの見通しを良くするため、必要に応じて Tailwind CSS（既に入っていれば）や Vanilla CSS のモジュールを作成し、クリーンなReactコードを保ってください。
- **テストプレビュー**
  開発中はローカル（先生のMac）で `npx remotion studio` を起動しながら、プレビュー画面でリアルタイムにデザインを推敲してください。（※本番の重い一括レンダリングは AWS M7i-flex インスタンス上で行うため、ローカルでの試作は遠慮なく行って大丈夫です）
